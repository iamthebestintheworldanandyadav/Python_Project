# Python_Project

🪔 Diwali Sales Analysis
- Analyzed Diwali sales data using Python to identify customer purchasing behavior and sales trends.

## 🛠️ Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Analysis by State, Gender, and Age Group
- Data Visualization using Charts

## 📈 Key Insights
- Uttar Pradesh, Maharashtra, and Karnataka had the highest orders
- Married women aged 26–35 contributed most to sales
- Food, Clothing, and Electronics were top-selling categories

## 📂 Dataset
- `Diwali Sales Data.csv`
